üÜò TROUBLESHOOTING:
--------------------------------------------------------------------------------

  Problem: "node: command not found"
  Solution: Install Node.js from https://nodejs.org/

  Problem: "python: command not found"
  Solution: Install Python from https://www.python.org/
           (Mac/Linux: use python3 instead of python)

  Problem: Servers won't start
  Solution: 1. Close all terminal windows
            2. Run setup script again
            3. Make sure no other app is using ports 8081 or 8000

  Problem: Camera not working
  Solution: 1. Allow camera permission in browser
            2. Position yourself 2-3 feet from camera
            3. Make sure camera sees your face + hands + chest

  For more help, see:
  - INSTALLATION.md  (detailed installation guide)
  - SETUP_INSTRUCTIONS.md  (camera setup guide)
  - README.md  (full documentation)
